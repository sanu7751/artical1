Introduction

I developed my first Machine Learning project, a Voice Recognition System, capable of accurately identifying 10-15 student voices.Here I used Deep Learning and Machine Learning Concept like MFCC (Mel-Frequency Cepstral Coefficients) and SVM (Support Vector Machine) to train the data module.the MFCC (Mel-Frequency Cepstral Coefficients) and SVM (Support Vector Machine) both are helped me to improve the accuracy of my project.

MFCC (Mel-Frequency Cepstral Coefficients) stands for Mel-Frequency Cepstral Coefficients.the MFCC (Mel-Frequency Cepstral Coefficients) used on both fields.it depending upon which type of project we develop.so here MFCC (Mel-Frequency Cepstral Coefficients) used for feature extraction.feature extraction means extract the raw data to a meaningful pattern that can be used on Machine Learning.In my project the MFCC (Mel-Frequency Cepstral Coefficients) is used for extracted the features from my audio data that should be I used on SVM (Support Vector Machine) model for classification.

Next is SVM (Support Vector Machine), SVM (Support Vector Machine) stands for Support Vector Machine. It’s an algorithm of Machine Learning. Generally it’s used for classification & regression type of task in Machine Learning.It is also called as a Supervised Learning. Because here we give Labeled Data(both input and output) then based on that data the Machine should predict. In my project when I give the extracted data into the SVM (Support Vector Machine), the SVM (Support Vector Machine) will separate each student voice for accurate prediction using a hyperplane. When a new voice is given, SVM (Support Vector Machine) compares the extracted MFCC (Mel-Frequency Cepstral Coefficients) features with the learned patterns and identifies the correct student.
